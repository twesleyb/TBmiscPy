# TBmiscpy

Miscelaneous python utility functions.


### Installing python packages from github:
```
# pip install git+<repo-url>#egg=<package>
pip install git+https://github.com/twesleyb/TBmiscPy.git#egg=TBmiscPy
```

 Alternatively, clone the repo and then run:
```
sudo python setup.py install
```
